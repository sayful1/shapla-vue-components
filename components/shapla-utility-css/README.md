# Shapla Utility CSS

Apply helper classes to almost any element, in order to alter their style.

## Table of contents

- [Installation](#installation)
- [Usage](#usage)

# Installation

```
npm i shapla-utility-css
```

# Available Utility Classes
* Accessibility
    * Screen Readers
* Effects
    * Box Shadow
* Flexbox
    * Align Content
    * Align Items
    * Align Self
    * Display
    * Flex
    * Flex Direction
    * Flex Grow
    * Flex Shrink
    * Flex Wrap
    * Justify Content
    * Order
* Layout
    * Container
    * Visibility
    * Responsive Visibility
* Spacing
    * Margin
    * Responsive Margin
    * Padding
    * Responsive Padding
* SVG
