# shapla-signature-canvas

A simple HTML5 canvas based smooth signature drawing for Vue

## Table of contents

- [Installation](#installation)
- [Usage](#usage)

# Installation

```
npm i shapla-signature-canvas
```

# Usage

Add the component:

```js
import signatureCanvas from 'shapla-signature-canvas';

export default {
  name: 'Hello',

  components: {
    signatureCanvas
  },
  data(){
    return {
      imageData:'',
    }    
  }
}

```
